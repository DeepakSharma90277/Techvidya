🚀 QUICK START GUIDE
════════════════════════════════════════════════════════════════════════

1️⃣  GET API KEYS (5 min)
    • Tavily:  https://tavily.com          (sign up free)
    • Groq:    https://console.groq.com    (sign up free)

2️⃣  CONFIGURE .ENV (2 min)
    cd ai-agent
    cp .env.example .env
    # Add your API keys to .env file

3️⃣  INSTALL DEPENDENCIES (3 min)
    cd ai-agent
    pip install -r requirements.txt

4️⃣  START SERVICES (1 min)
    # Windows: Run from root folder
    .\start-dev.ps1
    
    # Or manually in 3 terminals:
    # Terminal 1: cd server && npm run dev
    # Terminal 2: cd ai-agent && streamlit run app.py
    # Terminal 3: npm start

5️⃣  ACCESS & TEST (2 min)
    • Frontend:  http://localhost:3000
    • AI Agent:  http://localhost:8501
    • Backend:   http://localhost:4000
    
    Click "AI Assistant" in navbar → Test features!
